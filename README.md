## Installation

```bash
git clone git@github.com:jatin-2k/rails-blog.git
cd rails-blog
bundle install
rails db:migrate
rails db:seed
rails server
```

## Usage

```bash
rails server
# Open a web browser and go to http://localhost:3000
```
